# Insufficient evidence

1. Refine retrieval at most once if a materially better query is available.
2. If evidence is still insufficient, answer only the supported portion.
3. Put every missing fact, unresolved conflict, or currency concern in `limitations`.
4. Explicitly say that the retrieved sources are insufficient for a documented legal conclusion.
5. Suggest the exact document, date, case fact, or jurisdiction detail needed next.

Never fill an evidentiary gap from general model memory. Never turn an auxiliary Dadrah answer into a legal rule.
